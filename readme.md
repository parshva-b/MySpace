To run:
```
node v12.14.1
yarn 1.21.1
```

To set up envirnoment
```
cd to this directory
yarn install
yarn watch
```

Add any package:
```
yarn add [package] (-D {Dev Dependency})
```

For Ejs syntax use index.js as template
